Проекты DMSearch: один из моих первых старых проектов на php/mysql. Реализует хранение информации с канувшего в лету сайта delphimaster.ru, полнотекстовый поиск с фильтрами по нескольким полям. Актуальная ссылка - https://delphimaster.net RWL/Rewlis/SyncBooker/syncplayer: группа проектов, связанных с концепцией "Читай пока слушаешь" (Read While you Listen). Во-первых, это демо на сайте https://xayam.ru/rewlis, где можно легко посмотреть что это такое. Простыми словами - это аудио-плеер, который содержит синхронизированную информацию из каталога, который имеет книги на двух языках в формате текста и аудио. Интерфейс подает это все в виде прокручивающегося текста синхронно с воспроизводимым аудио. Во-вторых, аналогичная реализация концепции для Windows и Android - https://github.com/xayam/syncbookerkivy, доступные релизы программ справа в разделе Releases/Выпуски. Так же есть более ранний проект https://github.com/xayam/syncplayer который больше не поддерживается. VideoBook: проект монтажа видеокниг. Создан как продолжение идей RWL в видеоформате.
