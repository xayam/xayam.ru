Инструменты Разработка/IT PyCharm: Основная IDE для разработки на python. Notepad++: Вспомогательный редактор для текстовых файлов. Visio: Любимый редактор для векторной графики, результат обычно экспортирую либо в PDF, либо в PNG. GIT: Стандарт для управления версиями исходного кода. FileZilla: FTP-клиент для заливки файлов на сервер. GIMP: Бесплатный аналог фотошопа. Blender: Для 3D-моделирования (изучаю в настоящее время). Telegram/WhatsApp: Для общения и связи с клиентами. ...
