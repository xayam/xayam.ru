XAYAM.RU :: Личная страничка "ксаяма"

  • ChessVariants

    • Dodecahedron

      Проект создания модели шахматных досок с фигурам-додекаэдрами (12-тигранниками), их 3D-печати и продажи. Доски, фигуры, кости предназначены для игры в шахматы и варианты шахмат.

    • Novac

      Новый календарь, основанный на 4-дневной недели и имеющий дополнительный нулевой месяц "новий". Разработан для графиков работы два через два, четыре через четыре и сутки трое дома.

    • HexSum

      (в разработке). Игра для двоих на шестиугольной доске.

  • Rewlis

    Проект, реализующий концепцию RWL (REad While you LISten), читай пока слушаешь. В каталоге несколько аудиокниг + текст книг на двух языках (русском и английском). Плеер позволяет синхронно слушать и читать книги на этих языках. Аналогичную реализацию для Windows и Android можно найти по ссылке SyncBooker

  • Healers

    (в разработке). Проект, направленный на генерацию целительных звуков (по типу звукотерапии). Теория и практика.
