Обо мне

Изначально сайт был страничкой-визиткой основателя, но во второй версии добавилось несколько разделов, каталог, электронная книга, планируется привязать к книге движки для демонстрации вариантов игр. Основой сайта стали форматы XML/XSL, в которых хранится весь контент и коды для преобразования в HTML для браузера.

Сейчас вектор моего внимания направлен на доработку электронной книги и заполнение сайта контентом в структурированном формате, развитие своего магазина на Озоне/Авито и бренда КСАЯМ.

Всецело надеюсь, что такая работа будет проделана не зря, все исходники выкладываю на GitHub. Хочу расширить ряд шахматных фигур и костей для игры в варианты шахматных игр, для этого у меня есть компьютер, как основной инструмент, и недавно купленный лазерный гравер для творчества, которое может быть весьма обширным, поскольку идей для реализации тоже хватает.

DMSearch

[бета-версия] DMSearch: Многие поисковые системы основаны на хорошо известных методах и алгоритмах, разработанных еще до появления Интернет. Общая задача информационного поиска в Интернет состоит в том, чтобы найти документы, соответствующие информационной потребности пользователя.

Еще десять лет назад проблем с поиском необходимой информации у пользователя Интернет не возникало. Однако, с течением времени ситуация изменилась и теперь мы даже не всегда понимаем, какой необходимо сделать запрос для поиска нужной информации. Данный ресурс представляет собой тематический поисковик по старым веткам известного форума delphimaster.ru. В системе реализован так называемый полнотекстовый поиск в логическом режиме, все возможности поисковой строки этого режима Вы можете прочитать здесь.

Предыдущая первая версия поискового движка хорошо себя зарекомендовала среди пользователей, но была не приспособлена к расширению. В текущей второй версии значительно переделана структура хранения и доступа к данным, также сохранена обратная совместимость с первой версией. Перейти на сайт.

Новак

[новый календарь] Novac: Календарь для тех, кто живёт по ритму — а не по расписанию. Представьте календарь, который не просто показывает даты, но помогает чувствовать время. Novac создан не для того, чтобы вы отмечали встречи или праздники — он для тех, кто работает по сменам, по циклам, по внутреннему ритму: четыре дня через четыре дня, сутки через трое, два через два, или любой другой повторяющийся график, где обычный календарь сбивает с толку.

Вместо привычной недели из семи дней Novac вводит четырёхдневный цикл, вшитый прямо в ткань времени. Каждый день помечен одним из четырёх типов — alpha, beta, gamma, delta — и этот ритм идёт без перерывов с декабря 2019 года и на десятилетия вперёд. Независимо от того, выходной сегодня или рабочий, праздник или будень — вы всегда видите, на каком такте вы находитесь. Это особенно ценно для медиков, пожарных, дежурных, водителей, охранников, IT-специалистов и всех, чья работа живёт по собственным часам.

При этом Novac полностью совместим с обычной неделей: понедельник, вторник, выходные — всё на месте. Просто теперь у вас есть ещё один слой — невидимый, но надёжный пульс, который помогает не терять ориентацию в череде смен.

Novac — это не инструмент контроля. Это опора. Поле, в котором ваш график обретает форму, а вы — покой в ритме.

Ревлис

RWL / Rewlis / SyncBooker / syncplayer: группа проектов, связанных с концепцией "Читай пока слушаешь" (Read While you Listen). Во-первых, это демо на сайте rewlis, где можно легко посмотреть что это такое. Простыми словами - это аудио-плеер, который содержит синхронизированную информацию из каталога, который имеет книги на двух языках в формате текста и аудио. Интерфейс подает это все в виде прокручивающегося текста синхронно с воспроизводимым аудио. Во-вторых, аналогичная реализация концепции для Windows и Android - syncbookerkivy, доступные релизы программ справа в разделе Releases/Выпуски. Так же есть более ранний проект syncplayer который больше не поддерживается.

Summer

[в разработке] Summer: стратегическая настольная игра для двух игроков, разворачивающаяся на квадратной или гексагональной доске, где каждая фишка обладает числовым «весом». Игроки поочерёдно перемещают свои фишки вперёд или вбок (только на клетку, занятую своей фишкой), стремясь достичь стартовой линии противника.

При взаимодействии фишек — будь то объединение союзных или столкновение с вражескими — применяются правила, основанные на сравнении весов: они могут складываться, вычитаться или взаимно уничтожаться. Особую глубину добавляют поля с эффектами «плюс» и «минус», изменяющие вес при входе.

Игра завершается, когда все фишки одного игрока достигают цели; победитель определяется по сумме весов на финальной линии.

Встроенные ИИ-стратегии — от простых тактик до сложных адаптивных систем, включая «Абсолярис» и «Секрет» — превращают Summer не просто в игру, а в поле для исследования баланса между силой, жертвой, пространством и смыслом. Демо.

ВолумЛиб

[в разработке] VolumLib: библиотека избранного с Либрусека. Только русские книги в формате FB2. Только избранные жанры: компьютеры, программирование, наука, фантастика и т.д. Установлен минимальный размер на книгу - 100 KB. Сделано для записи на MicroSD-карту размером от 128 GB (на данный момент около 95 GB, оставшееся место может быть использовано ОС или пользователем для других файлов). Формат FB2.ZIP совместим с приложением FBReader - как с версией для смартфона на Android и iOS, так и для ПК на Windows. Также для Windows подойдет добавленная программа MyHomeLib. Для других программ-каталогизаторов, возможно, также подойдет inpx-файл для импорта библиотеки. Состояние: 2025.10.18 (97.357 fb2 книг, 46.537 авторов, 16.785 серий книг). Скачать всё.
